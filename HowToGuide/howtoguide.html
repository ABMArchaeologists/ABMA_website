1: Introduction to Agent-based Modeling
In this tutorial you will learn what simulations and agent-based models are and how they can help you in archaeological research.

 1.1: Introduction to Simulation 10 min Learn about important concepts for ABMA, like modeling approaches, simulation, emergence and complexity Setup, Go, Modeling, Simulation, Emergence, Complexity, Bottom-up, Top-down
1.2: Introduction to ABM 10 min Learn what ABM is; Learn what ABM's main components (see concepts) are; Learn about applications of ABM in archaeology Agents, environment, behavior, time, scenarios, variables, patches
1.3: How ABM can help 10 min Learn how ABMs are used in archaeological research
1.4: Introduction to NetLogo Interface 15 min Learn the basics of the NetLogo Web Interface Interactive/Authoring mode
2: Beginning with NetLogo
In this tutorial you will learn the basics of NetLogo by making your first simulation on the Out of Africal dispersal of homo sapiens. You will work with basic NetLogo syntax and learn how to set up a simulation and visualize their outcomes.

 2.1: The Setup Procedure 15 min Learn to work with the NetLogo Web Interface; Learn some common primitives used in NetLogo; Learn to set up the initialization phase of a simple ABM Procedures, Buttons, Errors, Recompile, Primitives create-turtles; set; shape; size; setxy; random; clear-all; reset-ticks;
2.2: Changing the world 5 min Learn about NetLogo's world (dimensions, coordinates, origin) and how to alter it Edit View, Coordinates, Origin, World Wrap
2.3: The Go Procedure 15 min Learn how to build a simple go procedure/simulation loop; Learn about basic primitives used for agent behavior (movement) Go Procedure/Simulation Loop, Forever Buttons tick; ask; random; forward; right/left;
2.4: Creating Custom Procedures 15 min Learn how to implement custom procedures; Learn how to use simple if-statements Agent Types (the Observer), Color Codes, If-statements hatch; sprout; color; if; random-float;
2.5: Creating Variables 20 min Learn how to implement custom variables; Global variable; Local variable; Slider; Comments; Conditional statements; Agentset globals; let; with; neighbors; move-to; one-of;
2.6: Custom Environment 10 min Learn what extensions are and how to add them to NetLogo; Learn how to import external data Extensions; Importing fetch; import-a; count; turtles-here
2.7: Advanced Environment 25 min Learn how to create custom agent variables; Learn how to use more complex if-else statements; Learn how to add more complex agent-environment interaction Patch-specific variables die; min-one-of
2.8: Complex Behavior 15 min Learn how to implement different simulation scenarios Scenarios; Switch button in-radius
2.9: Creating Plots 5 min Learn how to make plots; Learn how to use plots to compare multiple simulation scenarios. Plotting; Pen

5: How to model
In this tutorial you will learn more about how to actually incorporate agent-based modelling in your research. This tutorial focusses less on programming in NetLogo and more on the model development process. You will learn about the different phases in modelling and how to export data from your models.

 5.1: How to Model 101 10 min Learn what the model development process is Model development process
5.2: The Conceptual Phase 15 min Learn what the conceptual phase of the model development process is; Learn what consitutes good and bad research questions for modelling; Learn how to pick the right modelling technique; Learn about the importance of properly conceptualizing your model Conceptual phase; Research questions; Methods; Scales and resolutions
5.3: The Technical Phase 20 min Learn what the technical phase of the model development process is; Learn about parametrisation; Learn how to design experiments; Learn how to analyse and interpret your model Technical phase; Verification; Inline documentation; Parametrisation; Sensitivity analysis; Experiment design; Stylised facts; Parameter sweep; Run; Random number generator; Data analysis
5.4: The Dissemination Phase 10 min Learn what the dissemination phase of the model development process is; Learn about (the importance of) publishing your models; Learn about replication Dissemination phase; Publication; ODD; Replication(/reproduction)
5.5: Exporting Data 20 min Learn how to export results from NetLogo using basic export primitives; Learn what the BehaviorSpace is; Learn how to analyse model output BehaviorSpace; Exporting; Input widget export-view; export-output; export-plot; export-all-plots; export-world; word
